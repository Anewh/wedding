## Разработка

Команды

```shell
npm install

npm run front
# или
npm run front:dev

npm run back
# или
npm run back:dev
```

После `npm run back` проект доступен по адресу localhost:3000

## Развертывание

### Сборка

```shell
npm i -g @nestjs/cli

npm run front:prod

npm run build
npm run back:prod
```

### Запуск

```shell
npm run back:prod
```
